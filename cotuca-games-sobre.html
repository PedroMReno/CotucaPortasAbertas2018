COMO FOI FEITO?
   Para o desenvolvimento deste site, utilizamos três aplicativos que, para a nossa concepção, eram ideais e fundamentais ao COTUCA GAMES a fim de adquirir este resultado. Para que este site tivesse um design agradável e uma ótima interface com o usuário, foi necessário o uso de um editor de texto, que no caso foi o Sublime Text, mas que poderia ser até mesmo o Bloco de Notas, e um editor de imagem, que foi o Photoshop, para a criação personagens, ícones etc. Esses aplicativos não foram tão importantes quanto o GitHub (imagem ao lado), já que este aplicativo permite a nós uma programação compartilhada com colaboradores. Desta forma, não precisamos realizar projetos complexos individualmente. Além disso, ele funciona como um armazenamento em nuvem onde todos os envolvidos terão acesso para editar e enviar. Isso previne o rompimento de arquivos que foram salvos em um pendrive, por exemplo.
